---

## Nesta aula 

### aritméticos
- + (somar)
- "-" (subtrair)
- "*" (multiplicação)
- / (divisão real)
- % (resto da divisão inteira)
- ** (potenciação)

### Precedencia de operadores
- divisão, multi, são prioridades.
- parenteses ainda mais
- depois soma e subtração


### atribuição simples

```
var n = 3
n = n + 4

#ainda mais simples
n +=4
# simplificando ainda mais:
n++
 ```
- **isso vai valer para outros operadores**
